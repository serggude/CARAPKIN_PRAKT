## Цикл 1

### Исходный код

```python
def homework_basic_1(n):
    a = [0] * n
    for i in range(1, n):
        a[i] = a[i-1] * 2 + i
```
### 1. развертывание цикла 
```python
i = 1: a[1] = a[0] * 2 + 1
i = 2: a[2] = a[1] * 2 + 2
i = 3: a[3] = a[2] * 2 + 3
i = 4: a[4] = a[3] * 2 + 4
```
### 2. зависимости между операциями 
* a[1] записывается на i=1, читается на i=2 → FLOW
* a[2] записывается на i=2, читается на i=3 → FLOW
* a[3] записывается на i=3, читается на i=4 → FLOW

### 3. Расстояние зависимостей 
* D = 2 − 1 = 1
* D = 3 − 2 = 1
* D = 4 − 3 = 1

### 4. Классификация цикла 
D = 1 — критический случай

### Вывод

Цикл имеет зависимости между соседними итерациями. Параллелизация в невозможна. Для распараллеливания потребуются преобразования алгоритма.

## Цикл 2

```python
def homework_basic_2(n):
    a = [0] * n
    for i in range(2, n):
        a[i] = a[i-1] + a[i-2]
```

### 1. Развертывание цикла
```python
i = 1: a[1] = a[0] * 2 + 1
i = 2: a[2] = a[1] * 2 + 2
i = 3: a[3] = a[2] * 2 + 3
i = 4: a[4] = a[3] * 2 + 4
```

### 2. Зависимости между итерациями 
* a[2] → a[3] (через a[i-1]) → FLOW
* a[3] → a[4] (через a[i-1]) → FLOW
* a[2] → a[4] (через a[i-2]) → FLOW

### 3. Расстояние

* a[2] → a[3]: D = 3 − 2 = 1
* a[3] → a[4]: D = 4 − 3 = 1
* a[2] → a[4]: D = 4 − 2 = 2

### 4. Классификация 
Минимальное расстояние D = 1 — критический случай

### Вывод
Несмотря на наличие зависимости с расстоянием D = 2, критическая зависимость с D = 1 делает прямую параллелизацию невозможной. Цикл требует сложных преобразований.

## Цикл 3

```python
def homework_basic_3(n):
    a, b = [0] * n, [0] * n
    for i in range(1, n):
        a[i] = b[i-1] + 1
        b[i] = a[i-1] * 2
```

### 1. Развертывание цикла 
```python
i = 1:
    a[1] = b[0] + 1
    b[1] = a[0] * 2

i = 2:
    a[2] = b[1] + 1
    b[2] = a[1] * 2

i = 3:
    a[3] = b[2] + 1
    b[3] = a[2] * 2

i = 4:
    a[4] = b[3] + 1
    b[4] = a[3] * 2
```
### 2. Зависимости 
* b[i−1] → a[i] → FLOW
* a[i−1] → b[i] → FLOW

Перекрёстные зависимости между массивами a и b.

### 3. Расстояние
* b[1] → a[2]: D = 2 − 1 = 1
* a[1] → b[2]: D = 2 − 1 = 1
* аналогично для последующих итераций

### 4. Классификаия 
D = 1 — критический случай

### Вывод
Цикл содержит перекрёстные зависимости между массивами a и b с расстоянием D = 1. Итерации жёстко связаны, параллелизация невозможна без изменения структуры алгоритма.
